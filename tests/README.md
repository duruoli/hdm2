# Tests

This directory contains all test files for the project.

## Test Files

- `test_external_envs_integration.py` - Integration tests for external environments
- `test_gymnasium_metaworld.py` - Tests for Gymnasium and MetaWorld integration
- `test_hdm_training.py` - Tests for HDM training functionality
- `test_metaworld_hdm_training.py` - Tests for MetaWorld with HDM training
- `analyze_adroit_obs.py` - Analysis script for Adroit observations

## Running Tests

```bash
# Run all tests
python -m pytest tests/

# Run a specific test file
python tests/test_hdm_training.py
```
